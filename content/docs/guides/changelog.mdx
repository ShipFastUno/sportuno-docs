## v1.0.0 — 2026-03-24

Initial release of the Sportuno operator integration.

### Widget Session

- Create authenticated and anonymous widget sessions
- Configurable session expiry (300–3600 seconds)
- Optional `tournament_id` scope for focused sessions
- `allowed_origins` enforcement
- `sk_*` / `pk_*` key-pair model with separate test and live environments

### JavaScript SDK

- `window.Sportuno.init()` — embed the tournament widget
- `window.Sportuno.updateSession()` — push refreshed session token
- `window.Sportuno.closeSession()` — log out the widget
- `window.Sportuno.setLanguage()` — change language at runtime

### Wallet Callbacks

- Debit player wallet (tournament entry, rebuy, add-on)
- Credit player wallet (prize payout, refund)
- HMAC-SHA256 request signing with timestamp replay protection
- Idempotency key support on all financial operations
- 3-attempt retry with exponential backoff

### Webhooks

- `tournament.joined` — player enrolled in a tournament
- `tournament.settled` — tournament results finalized
- `tournament.cancelled` — tournament cancelled with refunds
