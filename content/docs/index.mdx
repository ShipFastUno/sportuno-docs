Integrating Sportuno into your platform takes three steps:

1. **Create a widget session** — Your backend calls our API with your secret key to get a session token
2. **Embed the widget** — Load our JavaScript SDK and pass the session token to render the tournament UI
3. **Handle wallet callbacks** — Implement endpoints for us to debit/credit your players' wallets

The widget handles everything else: tournament browsing, enrollment, betting, rebuy/add-on, and leaderboards.
